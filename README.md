+ 自我介绍（/intro）

+ 查询国内天气信息（/weather \<address> \<now/all>）

    <font color="gray" size=2><i>前一个变量可以写中文地名（只能写一级行政区，例如**丰台区**不能写成**北京市丰台区**），也可以写六位数字地址码；后一个变量表示查询当前天气（now）还是当日天气和未来三天的天气预报（all）</i></font>
